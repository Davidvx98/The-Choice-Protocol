# 🔒 Security Policy — The Choice Protocol

## Versiones soportadas

| Versión | Soporte de seguridad |
|---------|---------------------|
| 1.0.x   | ✅ Activo            |

## Reportar una vulnerabilidad

Si encuentras una vulnerabilidad de seguridad, **no abras una issue pública**.

En su lugar:
1. Abre un [Security Advisory](https://github.com/Davidvx98/The-Choice-Protocol/security/advisories/new) privado
2. O envía un email al mantenedor del proyecto

Responderemos en un máximo de 48 horas.

## Scope de seguridad

Las siguientes áreas son de especial interés:

- Exposición de API keys en el cliente
- Inyección de contenido en el prompt de Gemini
- Llamadas a APIs externas no autorizadas
- Valores de inputdel usuario sin validar que lleguen a APIs externas
